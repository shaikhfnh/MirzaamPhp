# Mirzaam Expo Website Redesign 2025/2026

## Project Overview
Redesign of https://mirzaam.com/ inspired by the premium, modern aesthetic of INDEX Exhibition.  
Dark, futuristic, luxurious theme with Arabic elegance.

**Goal**: Create a fast, smooth, premium single-page feeling website that is easy to convert to PHP later.

### Tech Stack
- HTML5 + Tailwind CSS (via CDN for development)
- Custom CSS + Keyframes for advanced animations
- Vanilla JavaScript
- Font: **Alexandria** (can be changed later)
- Bilingual: English + Arabic (RTL support)

### Color Palette
- **Primary**: `#232A42` (Dark Navy)
- **Secondary**: `#F4B223` (Gold Accent)
- **Background**: `#0A0F1C` or `#111827` (Dark)
- **Text Light**: `#F3F4F6`
- **Accent Teal** (optional): `#0F766E`
